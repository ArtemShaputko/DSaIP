# Цифровая обработка сигналов и изображений
## Лабораторная работа № 1. Преобразование Фурье

В качестве *музыкальных форматов* рекомендуется использовать WAV или PCM.

### Контрольные вопросы

1. Ортогональные функции. Ортонормированные функции. Базис ортогональных функций.
2. Ряд Фурье.
3. Семейство преобразований Фурье.
4. Дискретное преобразование Фурье (ДПФ).
5. Основные свойства дискретного преобразования Фурье.
6. Области применения дискретного преобразования Фурье.
7. Быстрое преобразование Фурье (БПФ).
8. Алгоритм БПФ (FFT) с прореживанием по времени (decimation-in-time, DIT).
9. Алгоритм БПФ (FFT) с прореживанием по частоте (decimation-in-frequency, DIF).
10. Преобразование вещественных и мнимых компонент в амплитуду и фазу.

### Цель работы

Изучение преобразования Фурье и его основных свойств, а также методики получения быстрого преобразования Фурье (БПФ).

### Исходные данные

Дана система функций $U = \{U_i(t)\}$, где $i \in [1, 2]$:

- $U_1(t) = a_1 \cdot \sin^{b_1}(v_1 \cdot t + \varphi_0)$;
- $U_2(t) = a_2 \cdot \cos^{b_2}(v_2 \cdot t + \varphi_0)$.

Коэффициенты $a_1$ = 0.5, $a_2$ = 1.0, $b_1$ = 3, $b_2$ = 5, $v_1$ = 2, $v_2$ = 4 и $\varphi_0$ = 1.5 заданы по варианту.

### 2 Выполнение работы

#### 2.1 Доказательство ортогональности системы функций

Доказать что заданная система функций $U$ образует базис ортогональных функций на интервале $[\varphi_0, \varphi_0 + 2\pi]$.

Продемонстрировать доказательство на защите лабораторной работы!

Уметь доказывать (опровергать) утверждение об ортогональность простейших систем (множеств) функций (на базе $sin$ и $cos$) на *произвольных* интервалах.

#### 2.2 Генерация сигнала $f(t)$

Сигнал задан функцией $f(t) = U_1(t) + U_2(t)$.

Построить построить график функции $f(t)$ при $t \in [0, D]$, где $D$ = 30 - длительность сигнала (задана по варианту).

Дискретизировать сигнал $f(t)$ в расчёте $d$ = 512 точек на каждые $2\pi$ секунды длительности сигнала (задано по варианту).

Полученный сигнал записать в *музыкальный файл*.

В дальнейшем работать только с этим дискретным сигналом.

#### 2.3 Дискретное преобразование Фурье (ДПФ)

Произвести преобразование.

Замерить время выполнения преобразования для сигнала $f(t)$.

Построить АЧХ функции $f(t)$.

#### 2.4 Быстрое преобразования Фурье (БПФ)

Аналогично подразделу 2.3.

#### 2.5 Обратное дискретное преобразование Фурье (ОДПФ)

Произвести обратное преобразование.

Замерить время выполнения преобразования для сигнала $f(t)$.

Построить график функции.

Полученный сигнал сохранить в музыкальный файл.

#### 2.6 Обратное быстрое преобразование Фурье (ОБПФ)

Аналогично подразделу 2.5.

#### 2.7 Оценка эффективности быстрого преобразования Фурье (БПФ)

Рассчитать эффективность работы БПФ для заданного количества точек $N = d$ (теоретически). Привести рассчёт для своего количества точек!

Сравнить время выполнения из подразделов 2.3 и 2.4.

Сравнить время выполнения из подразделов 2.5 и 2.6.

Сделать выводы и вынести их в раздел 3 (см. шаблон оформления отчёта).
